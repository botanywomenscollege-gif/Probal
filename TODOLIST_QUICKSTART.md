# 🚀 Quick Start Guide - Todo List Manager

## Download the .exe

1. Go to **GitHub Releases**: https://github.com/botanywomenscollege-gif/Probal/releases
2. Find the latest release with `TodoList.exe`
3. Click download
4. Save it to your desired location

## Installation

### Option A: Simple Install (Recommended)
```
1. Download TodoList.exe
2. Double-click to run
3. Done! No installation needed
```

### Option B: Create Desktop Shortcut
```
1. Right-click TodoList.exe
2. Send to → Desktop (create shortcut)
3. Double-click the shortcut anytime to launch
```

## First Time Setup

**First launch takes 5-10 seconds** (one-time only)

After launch, you'll see:
- Empty task list
- Input field to add tasks
- Category dropdown (Work, Personal, Shopping, Health, Other)
- Date picker for due dates
- A `todos.json` file created in the same folder

## Basic Usage

### Adding a Task
```
1. Type task name in the input field
2. Select a category from dropdown
3. Pick a due date
4. Press Enter or click "Add Task"
```

### Managing Tasks
```
Double-click task → Mark as complete/incomplete
Select task + Press Delete → Remove task
Type in search box → Find tasks
Use filter dropdown → View by category or status
```

### Data Backup
```
Regular backups of todos.json:
- Copy todos.json to a USB drive
- Email it to yourself
- Cloud storage backup
```

## Common Tasks

### How do I delete the app?
```
Simply delete TodoList.exe and todos.json
(Backup todos.json first!)
```

### How do I backup my tasks?
```
Copy todos.json to another location or cloud storage
```

### Can I move the app to another computer?
```
Yes! Copy both TodoList.exe and todos.json together
```

### What if I lose my tasks?
```
Restore from your todos.json backup
Place it in the same folder as TodoList.exe
```

### How do I clear completed tasks?
```
Click "Clear All" button to remove all completed tasks
```

### Can I edit tasks directly?
```
Yes! Open todos.json with Notepad to edit raw JSON
(Restart the app after editing)
```

## Troubleshooting

### App won't start
- ✓ Ensure Windows 10 or later
- ✓ Antivirus may block it (whitelist the .exe)
- ✓ Try running as Administrator (right-click)

### Tasks not saving
- ✓ Check folder write permissions
- ✓ Move app to Documents or Desktop folder
- ✓ Disable antivirus temporarily

### App runs slow
- ✓ Close other programs
- ✓ Delete old completed tasks
- ✓ Reduce total number of active tasks

### "Cannot find todos.json"
- ✓ Create new tasks (file auto-creates)
- ✓ Restore from backup
